
Used python library
--------------------
Need to install this two libs for geolocation. Don't forget to switch to the root user before installing, as you need to run traceroute using the root user mode.
	pip install geopy
	pip install geocoder


Used tools to Monitor
----------------------
To monitor the real time traffic, iftop tools can be used.
	sudo apt-get install iftop
	sudo iftop

To monitor real time cpu usage by running processes,
	top
